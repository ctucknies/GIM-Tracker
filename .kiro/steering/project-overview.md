# OSRS Player Tracker — Project Overview

This is a local web app that displays Old School RuneScape player stats and collection log data for the player **Drykeys** (previously known as `QQQ 420 Call` and `BuyAnthropic`).

## Architecture

A zero-dependency local app: static HTML/CSS/JS served by a Node.js HTTP server.

- `index.html` — Page structure with tabbed layout
- `styles.css` — Dark theme with OSRS-inspired gold accents
- `app.js` — Fetches data and renders the UI
- `server.js` — Simple Node.js static file server (port 8080)

## Data Pipeline

### Live API data (fetched on page load)
- **Wise Old Man API** (`https://api.wiseoldman.net/v2/players/drykeys`) — Skills, boss KC, activities, achievements
- Player username: `drykeys`

### Pre-built data files (generated by scripts)
- `collection_log.json` — The main collection log data with obtained items, dates, and prices
- `loot_data.json` — Parsed RuneLite loot tracker entries (intermediate)
- `wiki_clog_obtained.json` — Parsed wiki MHTML export showing obtained/missing items (intermediate)
- `clog_items_wiki.json` — Complete list of all collection log items from the OSRS Wiki

### Build scripts
- `parse_loot.js` — Parses RuneLite loot tracker from profile properties
- `parse_wiki_clog.js` — Parses saved wiki collection log table MHTML
- `build_collection.js` — Combines all sources into final `collection_log.json`
- `fetch_collection_log.js` — (Legacy) Attempted collectionlog.net API fetch
